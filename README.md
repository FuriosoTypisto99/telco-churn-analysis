# Telco Customer Churn Analysis (SQL)

Analyzed 7,000+ telecom customer records to identify key churn drivers and high-risk customer segments.

## Key Insights
- Month-to-month customers have the highest churn
- New customers (<6 months) are most likely to leave
- High-paying customers show higher churn rates
- Electronic check users have higher churn

## Tools Used
- SQL (MySQL)
